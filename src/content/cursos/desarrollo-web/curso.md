# Desarrollo Web

Bootcamp intensivo diseñado para que un principiante salga con un proyecto web funcional y entienda los fundamentos de cómo se construye, versiona y modifica software con ayuda de IA.

## Metodología

**Aprendizaje en espiral y Vibecoding.** En cada sesión usaremos IA para generar la base estructural y nos enfocaremos en entender, conectar y modificar el código. La IA escribe mucho por nosotros, pero tú decides dónde va cada cosa.

**Trabajo autodidacta intensivo.** La sesión es para construir e integrar; las tareas entre sesiones son para desarmar, analizar y reflexionar.

## Lo que vas a aprender

- Configurar un entorno de desarrollo profesional (Node.js, VS Code, terminal).
- Escribir HTML, CSS y JavaScript moderno con la asistencia de IA.
- Iterar con Vibecoding: promptear, leer el código que devuelve la IA, modificarlo.
- Versionar tu trabajo con Git: commits, status, branches.
- Integrar todo en un mini-proyecto publicable.

## Estructura del bootcamp

| # | Sesión | Estado | Duración |
|---|--------|--------|----------|
| 1 | [Fundamentos, Hola Mundo y Vibecoding](./sesion-1/sesion.md) | Disponible | 3 h |
| 2 | Componentización y Multiplicación | Planificada | 3 h |
| 3 | Layouts y Rutas | Planificada | 3 h |
| 4 | Interactividad y Refinamiento | Planificada | 3 h |

### Sesión 1 — Fundamentos, Hola Mundo y Vibecoding (disponible)

Pasar de una computadora en blanco a una página web real renderizada en el navegador con un diseño base atractivo. Introducir Vibecoding como metodología y Git como workflow profesional.

### Sesión 2 — Componentización y Multiplicación (planificada)

Construir un grid de tarjetas reutilizando un componente `Card.astro`. Introducir **Props**, el modelo de caja real (`padding`, `margin`, `border-radius`) y los primeros **frontmatters** en Markdown.

### Sesión 3 — Layouts y Rutas (planificada)

Pasar de páginas sueltas a un sitio web con navegación y múltiples vistas. Crear un `Layout.astro` con `<slot />`, entender el enrutamiento automático de Astro, estados interactivos con Tailwind (`hover:`, `focus:`) y empezar a trabajar con **ramas** en Git.

### Sesión 4 — Interactividad y Refinamiento (planificada)

Pulir detalles visuales, agregar interactividad ligera con `<script>` y dejar el proyecto publicable. Tocar dark mode con `dark:`, MDX (Markdown + componentes) y la fusión de ramas con `git merge`.

## Audiencia

Personas con conocimientos básicos de HTML que quieran dar el salto a JavaScript y al desarrollo web moderno, usando IA como copiloto de aprendizaje. Si nunca escribiste una etiqueta `<div>` a mano, este curso te lleva de cero a un portafolio local versionado con Git.
